## Installation

#### 1. Clone the repository .

```bash
git clone https://github.com/panhboth111/node-email
```

#### 2. Install dependencies .

Navigate into the cloned repo and do the following
```bash
npm i
```
or
```bash
yarn
```

## Run the server

Navigate into the cloned repo and do the following
```bash
npm start
```
or for yarn 
```bash
yarn start
```
## How to use:

#### 1. Open postman.
#### 2. Make a get request to http://localhost:3000/
#### 3. See the result
